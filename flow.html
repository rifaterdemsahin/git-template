Step-by-step explanation

  1. Find the template — Browse GitHub and locate rifaterdemsahin/git-template. Enable Template repository in repo settings if not already done.
  2. Create your repo — Run gh repo create my-new-project --template rifaterdemsahin/git-template --public --clone to stamp all template files into a brand-new repository with a clean commit history.
  3. Open in VS Code — cd my-new-project and open the folder. The CI skeleton is already in place.
  4. Prime Copilot — Open Copilot Chat and paste the initial prompt (project name, tech stack, purpose, conventions). This gives Copilot full context from the very first message.
  5. Persist the prompt — Optionally save it to .github/copilot-instructions.md so every team member benefits automatically without any manual copy-pasting.
  6. Build — Copilot is now context-aware and ready to accelerate your development.
See also: Usage Guide for the gh repo create --template command details, and Copilot Initial Prompt for the ready-to-use prompt template.
