# git-template

A reusable CI skeleton — stamp it onto every new project in seconds.

## Quick start

```bash
gh repo create my-new-project \
  --template rifaterdemsahin/git-template \
  --public \
  --clone
cd my-new-project
```

The `--template` flag copies all files from this repository into a brand-new
repo with a clean commit history.  No more copy-pasting boilerplate.
